# Advanced Encode / Decode Tool

A simple web-based tool for encoding and decoding text using various formats including Base64, URL encoding, MD5, and SHA-256 hashing. It also supports case conversion and easy copy/clear functions.

## Features

- Base64 encode/decode
- URL encode/decode
- MD5 hashing
- SHA-256 hashing
- Uppercase and lowercase conversion
- Copy output to clipboard
- Clear input and output fields
- Responsive design for desktop and mobile

## Usage

1. Enter your text in the input area.
2. Click the desired action button to encode, decode, hash, or convert case.
3. The result will display in the output area below.
4. Use the **Copy** button to copy the output text.
5. Use the **Clear** button to clear both input and output.

## Installation

Simply open the `index.html` file in any modern web browser. No server setup required.

## Dependencies

- None (Uses built-in JavaScript APIs)
- MD5 hashing implemented with embedded JavaScript code
